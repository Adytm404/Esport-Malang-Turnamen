# Malang Turnamen (Wordpress)

A brief description of what this project does and who it's for


## Features

- News Article Post
- Form Registration
- User Friendly Interface
- Responsive Website
- Fast Loading

bash

bash

*Client:* PHP 7.4, Wordpress V6

*Plugins:* Elementor, Elementor Pro

*Server:* Alibaba Cloud VPS
